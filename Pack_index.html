module Pack_index: sig .. end
Pack indexes.
module Raw: sig .. end
type t
Abstract type of pack index.
val create : ?cache_size:int -> Cstruct.buffer -> t
Create a pack index object. The results of find_offset are cached for upto cache_size SHA objects
val find_offset : ?scan_thresh:int -> t -> SHA.t -> int option
find_offset searches the index for the offset of the SHA object. Binary search is performed until the candidates are narrowed down to scan_thresh or less.
val mem : t -> SHA.t -> bool
mem checks if the SHA object is contained in the index object
